PROJECT OVERVIEW
════════════════════════════════════════════════════════════════════════════════

A complete machine learning pipeline for predicting the pathogenic severity of 
FBN1 mutations using Support Vector Machines trained on DNA sequence k-mer 
features.

INPUT:   FBN1 mutation (position + nucleotide change in NM_000138.5)
OUTPUT:  Severity prediction with confidence scores

SEVERITY CLASSES
════════════════════════════════════════════════════════════════════════════════

1. Benign                          - Normal variation, not harmful
2. Likely benign                   - Probably normal variation
3. Uncertain significance          - Unknown clinical significance  
4. Likely pathogenic              - Probably disease-causing
5. Pathogenic                     - Confirmed disease-causing

MODEL SPECIFICATIONS
════════════════════════════════════════════════════════════════════════════════

Features:              K-mer counting (3-mers) + TF-IDF vectorization
Input dimension:       300 features
Classifier:            Support Vector Machine (SVM)
Kernel:               Radial Basis Function (RBF)
Classes:              5 severity levels
Training samples:     150 (synthetic - augmented from 5 original)
Cross-validation:     5-fold stratified
Performance:          ~33% test accuracy (synthetic data)

USAGE EXAMPLES
════════════════════════════════════════════════════════════════════════════════

CLI: Single mutation
  $ python predict_mutation.py --position 8606 --base C

CLI: JSON output
  $ python predict_mutation.py --position 8606 --base C --json

Python: Simple prediction
  from predict_mutation import PredictionPipeline
  pipeline = PredictionPipeline()
  result = pipeline.predict(8606, 'C')
  print(result['prediction'])  # 'Benign'
  print(result['confidence'])  # 0.1578

Python: Batch predictions
  mutations = [(8606, 'C'), (8595, 'C'), (1000, 'A')]
  results = [pipeline.predict(pos, base) for pos, base in mutations]

Bash: Process multiple mutations
  for mutation in 8606,C 8595,C 1000,A; do
    pos=$(echo $mutation | cut -d, -f1)
    base=$(echo $mutation | cut -d, -f2)
    python predict_mutation.py --position $pos --base $base
  done

TROUBLESHOOTING
════════════════════════════════════════════════════════════════════════════════

Error: FileNotFoundError: svm_mutation_classifier.pkl
→ Ensure you're in the directory with all .pkl files and .fasta file

Error: Position out of range
→ Position must be 1-indexed and between 1 and 11609

Error: Reference already has C
→ New base must differ from the reference base at that position

Low accuracy
→ Current model uses synthetic data. Provide real labeled data for retraining.

Memory error on large batches
→ Process mutations in chunks of 1000 at a time

SUPPORTED MUTATIONS
════════════════════════════════════════════════════════════════════════════════

Position range:  1 to 11,609 (NM_000138.5 transcript)
Bases:           A, C, G, T
Type:            Single nucleotide variants (SNVs)
Format:          1-indexed cDNA notation (e.g., c.8606T>C)

NOT SUPPORTED:
  • Insertions or deletions
  • Copy number variations
  • Structural variants
  • Multiple simultaneous mutations
